# **Tensor Build**

Description du projet:

**Le problème scientifique abordé :** Les matériaux sont constamment soumis à des contraintes thermiques, mécaniques et environnementales, mais ces dernières sont généralement imperceptibles à l’œil nu. Pour aider les étudiants à comprendre et visualiser les effets des stress constants des matériaux, nous proposons le projet Tensor Build.

**La solution informatique propose :** Une simulation interactive, mais aussi visuelle et éducative. Le tout soutenue par Python, SciPy et NumPy. Python nous permet de coder ; c’est notre langage pour faire fonctionner la simulation. Tandis que SciPy et NumPy, c’est plus pour tout ce qui est calcul, interaction d’objets dans l’environnement, stress sur les matériaux.

**Les utilisateurs visés :** Le public ciblé est un jeune public ayant été exposé au physique de base, mais voulant pousser plus loin et pouvoir visualiser ce qui ne se voit pas au niveau des matériaux. Ces personnes se retrouvent généralement en fin d’études au cégep ou à l’université.

Comment faire rouler le projet:
```python
# On bâtit l'environnement
python venv venv
python source/bin/activate
pip install -r requirements.txt

# Rouler le fichier d'application principal
python main.py
```
